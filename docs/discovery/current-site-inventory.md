## Проблемы текущего сайта

- Доменная и SEO-логика неактуальна: canonical ведет на `http://dtcnab.ru`, новый домен должен быть `mos-disel.ru`.
- Цена конфликтует: в meta - `от 35 руб./л`, в hero - `48 р/л`.
- Есть технический/черновой текст `create your own block from scratch`.
- Есть опечатки и шероховатости: `поберем`, `заказачиков`, `НОБХОДИМЫЕ`, `д оставку`, `нефтебазепломбируется`, `точность` разбита пробелом, `Metalica`.
- Тексты перегружены и местами написаны от разных лиц: `мы`, `наша компания`, `Вы`.
- Слишком много разрозненных CTA и повторяющихся форм.
- Сайт визуально наследует Tilda-стиль 2019-2024 и выглядит устаревшим для B2B-поставщика.
- Не хватает аккуратной структуры для юридических клиентов: документы, НДС, договор, объемы, логистика, сроки, качество.
- Не хватает понятного маршрута для частного клиента: объем, сезонность, хранение, доставка, оплата.
