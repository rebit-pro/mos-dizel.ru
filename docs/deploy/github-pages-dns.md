# GitHub Pages и DNS для mos-dizel.ru

## Текущая схема

- Репозиторий: `rebit-pro/mos-dizel.ru`
- Рабочая папка WSL: `/home/user/mos-dizel.ru`
- Домен: `mos-dizel.ru`
- Деплой: GitHub Actions -> GitHub Pages
- Файл домена в проекте: `public/CNAME`

## Что внести в Руцентр

На скрине открыт экран управления DNS-серверами. GitHub Pages не требует менять NS на серверы GitHub: нужно, чтобы у домена была DNS-зона, где можно редактировать ресурсные записи.

1. В Руцентре открой домен `MOS-DIZEL.RU`.
2. На экране `Управление DNS-серверами` оставь DNS у Руцентра, если текущий режим `Хостинг` дает доступ к редактированию DNS-зоны.
3. Если в режиме `Хостинг` нельзя редактировать записи, переключи услугу на `DNS-master Anycast`, сохрани изменения и дождись применения.
4. Перейди в управление DNS-зоной / ресурсными записями домена.
5. Удали старые записи для корня домена и `www`, которые ведут на старый хостинг/парковку.
6. Добавь записи для корневого домена:

| Имя | Тип | Значение |
|---|---|---|
| `@` или пусто | `A` | `185.199.108.153` |
| `@` или пусто | `A` | `185.199.109.153` |
| `@` или пусто | `A` | `185.199.110.153` |
| `@` или пусто | `A` | `185.199.111.153` |

7. Добавь запись для `www`:

| Имя | Тип | Значение |
|---|---|---|
| `www` | `CNAME` | `rebit-pro.github.io.` |

8. TTL можно оставить стандартный, например `3600`.
9. Не добавляй wildcard-запись `*`.

После сохранения DNS может обновляться до 24 часов.

## Как проверить DNS

В WSL:

```bash
dig mos-dizel.ru +noall +answer -t A
dig www.mos-dizel.ru +noall +answer -t CNAME
```

Ожидаемо для корня должны появиться IP GitHub Pages:

```text
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

## Что настроить в GitHub

1. Открой репозиторий `https://github.com/rebit-pro/mos-dizel.ru`.
2. Перейди в `Settings -> Pages`.
3. В `Build and deployment` выбери `Source: GitHub Actions`.
4. В `Custom domain` укажи `mos-dizel.ru` и нажми `Save`.
5. Дождись проверки DNS.
6. Когда станет доступно, включи `Enforce HTTPS`.

В проекте домен уже указан в `public/CNAME`, а GitHub Pages settings уже выставлены на `mos-dizel.ru`.

## Локальный запуск

```bash
cd /home/user/mos-dizel.ru
make install
make dev
```

По умолчанию dev-сервер открывается на `http://127.0.0.1:5173/`.
